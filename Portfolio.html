Personal Projects

Covid-19 Tracker:

Tracks Covid-19 cases, recoveries, and deaths for every country on a simple web page that includes a map, graph, and chart. It was created by using JavaScript React, and the material UI package for the front-end interface.

Flight Tracker:

A real time program that lets you track the destination of any flight in America and see all of the nearby eating and lodging places within a certain mile radious. I did this project with a team for our TAMU hack 2020 project. I personally took this opportunity to learn how to use API's and further improve my teamwork skills in a intense environment.

Black Jack Game:

A player vs. computer blackjack game that offers a simple interface for card images, sounds, a table to keep track of wins/loses/draws and a money component. I used this program to help me learn HTML, CSS, and Javascript.
